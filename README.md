# Telegram MCP Server

This is a Model Context Protocol (MCP) server that enables Telegram integration with Poke.

## Overview

The Telegram MCP server provides a bridge between Telegram's messaging platform and Poke, allowing AI assistants and applications to interact with Telegram through the standardized MCP interface.

## Features

- **Telegram Integration**: Connect to Telegram's API through MCP
- **Poke Compatible**: Designed to work seamlessly with Poke
- **Standardized Protocol**: Built on the Model Context Protocol for interoperability
